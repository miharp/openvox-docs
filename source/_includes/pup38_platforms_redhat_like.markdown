Puppet publishes and tests open-source Puppet 3.8 packages for the following versions of Red Hat Enterprise Linux (RHEL) to [yum.voxpupuli.org](https://yum.voxpupuli.org):

-   Enterprise Linux 7
-   Enterprise Linux 6
-   Enterprise Linux 5
-   Enterprise Linux 4

This also applies to any distributions that maintain binary compatibility with RHEL, including but not limited to CentOS, Scientific Linux, and Oracle Linux.
